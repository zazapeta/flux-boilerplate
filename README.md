# flux-boilerplate
you need : 

    • Python 3
    • npm
    
Use like this : 

    • Open startFlux.py
    • Modify APP_NAME
    • cd into/your/dir
    • copy startFlux.py into/your/dir
    • python startFlux.py
    • start your server
    
python flux mini-boilerplate (inspired by [todomvc](https://github.com/facebook/flux/tree/master/examples/flux-todomvc/ "flux-todomvc") from facebook repo)
